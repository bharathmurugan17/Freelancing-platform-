# рџ“– Overview

**Freelancing Platform** is a modern web application that connects **clients** and **freelancers** in a secure digital marketplace. The platform enables clients to publish projects, receive competitive proposals, hire skilled professionals, and efficiently manage project workflows.

Freelancers can create professional profiles, discover opportunities, submit proposals, collaborate with clients, and build their reputation through completed projects.

The application follows modern full-stack development practices with responsive UI, secure authentication, scalable backend architecture, and RESTful APIs.

# вљ™пёЏ Installation

## Clone Repository

```bash
git clone https://github.com/bharathmurugan17/Freelancing-platform-.git
```

## Navigate to Project

```bash
cd Freelancing-platform-
```

## Install Dependencies

```bash
npm install
```

## Configure Environment Variables

Create a `.env` file in the project root.

```env
PORT=5000

MONGODB_URI=your_mongodb_connection_string

JWT_SECRET=your_jwt_secret

CLIENT_URL=http://localhost:3000
```

---

## Start Backend

```bash
npm run server
```

## Start Frontend

```bash
npm run dev
```
